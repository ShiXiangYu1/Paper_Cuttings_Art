# 剪纸学习网站

这是一个帮助用户系统性学习剪纸艺术的网站项目。

## 项目结构
```
├── index.html              # 主页面
├── pages/                  # 各个功能页面
│   ├── basic/             # 基础教程
│   ├── advanced/          # 进阶教程
│   ├── gallery/           # 作品展示
│   └── community/         # 社区互动
├── assets/                # 静态资源
│   ├── images/           # 图片资源
│   ├── videos/           # 视频资源
│   └── icons/            # 图标资源
├── styles/               # 样式文件
│   ├── main.css         # 主样式文件
│   └── themes/          # 主题相关样式
└── scripts/             # JavaScript文件
    └── main.js          # 主脚本文件

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- 字节跳动CDN加速

## 浏览器支持
- Chrome (推荐)
- Firefox
- Safari
- Edge

## 开发指南
1. 所有页面采用模块化开发，每个功能页面独立文件夹
2. 样式采用BEM命名规范
3. 使用CSS变量管理主题
4. JavaScript采用ES6+标准
